# 上海海事大学校园网自动认证工具-快速入门

您好，欢迎使用上海海事大学校园网自动认证工具。

## 介绍

本工具是一个用于自动认证上海海事大学校园网的工具，可以帮助你快速连接校园网。
此外，还可以自动检测网络连接状态，当网络断开时自动重新连接。

## 支持平台

由于程序使用Python编写，因此支持**全平台**！

- Windows
- Apple macOS
- Linux
- Docker Image
